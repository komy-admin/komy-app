# 📱 Fork'it Application

Application mobile développée avec Expo et React Native.

## 🚀 Technologies utilisées

- [Expo](https://docs.expo.dev/) - Framework et plateforme pour React Native
- [React Native](https://reactnative.dev/docs/getting-started) - Framework pour le développement mobile
- [React Native Reusables](https://rnr-docs.vercel.app/getting-started/introduction/) - Librairie de composants React-Native
- [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) - Animations fluides et performantes
- [Redux](https://redux.js.org/) - Gestion de l'état global
- [Lucide React Native](https://lucide.dev/guide/packages/lucide-react-native) - Bibliothèque d'icônes

## 🛠 Installation

Node 22

```bash
# Installation des dépendances
npm install

# Démarrage du projet
npx expo start
```
